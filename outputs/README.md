# outputs

This directory stores experiment outputs downloaded from servers and local smoke-test outputs.

## Naming Rules

Official server outputs:

```text
phase<N>_<name>_server.json
phase<N>_<name>_server.md
log_phase<N>_<name>.txt
```

Local smoke outputs:

```text
phase<N>_<name>_local_smoke.json
```

Only `*_server.json` / `*_server.md` files can support paper-facing claims unless explicitly approved in `experiment_report.md`.
